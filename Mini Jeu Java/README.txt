Concept :
The game consist of a race between two players' creatures.

Rules :
	You become friend with creature by giving them an object. You collect objects in chest.
	After all turns have past, creature run each a distance according to their weight.

How to play :
	For parameters use ENTER to confirm parameters
	For the game : use ZQSD to move and SPACE to use your action

Jeu.java contains the mains
